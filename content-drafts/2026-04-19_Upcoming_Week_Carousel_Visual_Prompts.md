## Master design system (applies to every slide)

- **Dimensions:** 1080×1350 (4:5 portrait). LinkedIn carousel native.
- **Background:** near-black `#0B0B0B`.
- **Primary text:** off-white `#F5F1E8`.
- **Typography:**
  - Headline: editorial serif (Canela, GT Super, Tiempos Headline, or Playfair Display — pick one and hold it across all 9).
  - Subtitle / metadata: geometric sans in small caps, tracked +120 (Inter, GT America, or Söhne).
- **Accent color:** one accent per slide, used only for a single element (underline, rule, numeral, or dot). Never two accents on one slide.
- **Top-left corner (every slide):** small-caps carousel index — `01 / 09`, `02 / 09`, etc.
- **Bottom-left corner (every slide):** small-caps footer — `APR 20–26 · NYC · ALEX YEDI`.
- **Negative space:** ~40% empty. No decorative clutter. No stock photography. No 3D effects. No gradients. No icons. No emoji.
- **Rule:** if a slide needs more than one accent color or more than two type weights to land, the copy is wrong — rewrite the copy, not the design.

## Production notes

- **Tool recommendation:** Ideogram 2.0 or Imagen 3 for text-rendering accuracy. Midjourney v6 will struggle with the exact typography — use only if you're willing to retype in Canva/Figma after. ChatGPT image generation works for quick passes.
- **Iteration loop:** Generate all 9 at once with identical style prompts, then regenerate individual slides where text renders wrong. Keep the accent color and font family language identical across attempts so the set stays cohesive.
- **Manual cleanup:** For anything LinkedIn-destined, drop all 9 into a single Figma frame after generation to verify: (a) numbered badges are consistent, (b) footer string is identical on every slide, (c) font weights match across the set. AI image tools drift on these.
- **Alt text:** Paste the "Alt text" line from each slide into LinkedIn's alt-text field when uploading. Accessibility + LinkedIn's reach algorithm both reward it.
- **Ordering check:** Slide 04 (HockeyStack) runs Tue 1 PM, Slide 05 (Azure) runs Tue 2 PM, Slide 06 (Voice AI) runs Tue 5:30 PM — the carousel preserves chronological order top-to-bottom of the post.
